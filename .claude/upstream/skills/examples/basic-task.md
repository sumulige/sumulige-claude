# Basic Task Example

> 基础任务处理技能示例

**版本**: 1.0.0
**作者**: AI
**标签**: [example, basic]
**难度**: 初级

---

## 概述

这是一个基础的任务处理技能，展示如何创建简单的 Claude Code 技能。

## 适用场景

- 学习技能模板结构
- 作为创建新技能的起点
- 理解技能的基本组成

## 触发关键词

```
basic task, simple task, example task
```

## 使用方法

### 基础用法

在 Claude Code 中说：
```
用 basic-task 技能处理这个任务
```

### 技能响应流程

1. 分析任务需求
2. 确定所需步骤
3. 逐步执行
4. 验证结果

## 输出格式

```yaml
task:
  name: "任务名称"
  status: "completed"
  steps: ["步骤1", "步骤2", "步骤3"]
  result: "执行结果"
```

## 注意事项

- 保持简单
- 专注于单一目标
- 记录每个步骤
