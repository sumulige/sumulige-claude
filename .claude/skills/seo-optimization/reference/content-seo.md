# Content SEO

## 标题层级规则

```
H1: 页面唯一主标题（包含品牌/项目名）
  H2: 主要章节（功能关键词）
    H3: 子章节（具体特性）
      H4-H6: 细节层级
```

## 关键词策略

### 密度控制

- 核心关键词: 1-2%
- 次要关键词: 0.5-1%
- 长尾关键词: 自然出现

### 位置优化

1. **标题 (H1)**: 必须包含核心关键词
2. **首段**: 前 100 字内出现核心关键词
3. **小标题 (H2-H3)**: 包含相关关键词
4. **结尾**: 重申核心价值

## README 结构模板

```markdown
# {项目名}

{一句话价值主张，包含核心关键词}

[Badges]

## Quick Start

{3-5 行即可上手的代码}

## Features

- **{特性1}**: {描述}
- **{特性2}**: {描述}

## Installation

{安装命令}

## Usage

{基本使用示例}

## Documentation

{文档链接}

## Contributing

{贡献指南}

## License

{许可证}
```

## 文档链接优化

### 内部链接

- 使用描述性锚文本
- 避免 "点击这里"
- 链接到相关章节

### 外部链接

- 添加 `rel="noopener noreferrer"`
- 链接到权威来源
- 定期检查死链
